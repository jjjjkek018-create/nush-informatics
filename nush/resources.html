Ресурси для самостійного опрацювання

Корисні посилання, тренажери, відеоуроки та ідеї для проєктів

Ви завершили вивчення розділу «Опрацювання даних в електронних таблицях». Щоб поглибити свої знання та відпрацювати практичні навички, скористайтеся добіркою додаткових матеріалів. Тут зібрано онлайн-середовища для роботи з таблицями, інтерактивні вправи, відеопояснення складних тем, а також ідеї для власних міні-проєктів.

Онлайн-середовища для роботи з таблицями

Google Таблиці Безкоштовний хмарний табличний процесор. Дозволяє працювати спільно з однокласниками в реальному часі. Microsoft Excel Online Офіційна веб-версія Excel. Має майже всі функції настільної програми та не потребує встановлення. LibreOffice Calc Вільний та безкоштовний табличний процесор. Можна завантажити для Windows, macOS або Linux. OnlyOffice Ще один безкоштовний офісний пакет із потужним редактором електронних таблиць.

Ідеї для міні-проєктів

Сімейний бюджет Створіть таблицю доходів та витрат вашої родини за місяць. Використайте функції SUM, AVERAGE, побудуйте діаграму розподілу витрат за категоріями.
Табель успішності класу Змоделюйте таблицю з оцінками учнів з кількох предметів. Обчисліть середній бал кожного учня, визначте рейтинг, застосуйте умовне форматування для виділення відмінників.
Калькулятор калорій Складіть список продуктів з їхньою калорійністю. За допомогою формул реалізуйте підрахунок загальної калорійності обраних страв.
Погодний журнал Протягом тижня записуйте температуру повітря вранці та ввечері. Побудуйте графік зміни температури та обчисліть середнє значення за період.
